### How to run it on your own machine

1. Create Secrets in ~/.streamlit/secrets.toml

# .streamlit/secrets.toml

user="xxx"
password="xxx"
account="xxx"
host="xxx"
port="xxx"
warehouse="xxx"
role="xxx"
   
3. Install the requirements

   ```
   $ pip install -r requirements.txt
   ```

4. Run the app

   ```
   $ streamlit run streamlit_app.py
   ```
